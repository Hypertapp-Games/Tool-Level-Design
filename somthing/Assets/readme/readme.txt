Nhập chiều rộng và chiều cao và nhấn vào nút Export Size để gen ra các tile

2 hàng nút có màu
   - Các nút vuông: chứa thông tin của các normal tile
   - Các nút tròn: chứa thông tin của các hole tile
Cách dùng:  - Click vào nút (nút sẽ có hightlight màu vàng để feedback là nút đang được chọn)
            - Click vào các tile để thay đổi màu và hình dạng của tile tương ứng với nút đang được chọn 
            
Nhấn vào Nút Play để bật PlayMode và tắt EditMode (Ngược lại với EditMode)

Ở trong PlayMode sẽ chọn đầu và đuôi của con rắn để kéo đi
Nếu kéo con rắn vào lỗ thì nó sẽ biến mất

Nhấn vào nút Export để lưu dữ liệu thành dạng văn bản, file txt được lưu ở trong folder ScreenCapture


Chọn object ToolEditor trên scene, trong Inspector kéo xuống sẽ thầy file Path
Coppy đường dẫn của file text và dán vào file Path
Sau đó nhấn vào nút Import để load dữ liệu của file path vào màn hình
